# Machine-Learning-Monthly-Report
中学生科技创新后备人才培养计划，简称英才计划或中学生英才计划，是中国科学技术协会和中华人民共和国教育部自2013年开始共同组织实施的人才培养计划。本仓库内容记录了在为期一年的培养中我的收获。
